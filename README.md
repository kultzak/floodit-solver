# floodit

A game where the minimum number of flooding operations is NP-hard  for a number
of colors >= 3.

[The Complexity of Flood Filling Games.](https://arxiv.org/pdf/1001.4420.pdf)


# how to run
from src directory run
```
$ make
$ ./floodit
```
